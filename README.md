# Kiro Superpowers

An agentic development workflow for [Kiro CLI](https://github.com/kirolabs/kiro-cli): brainstorm → design → plan → TDD → subagent execution → review.

## Skills Included

| Skill | Description |
|-------|-------------|
| brainstorming | Structured ideation and requirements gathering |
| writing-plans | Break tasks into actionable plans |
| executing-plans | Step-by-step plan execution |
| test-driven-development | TDD workflow |
| systematic-debugging | Root cause analysis and fixes |
| subagent-driven-development | Parallel subagent task dispatch |
| dispatching-parallel-agents | Multi-agent coordination |
| requesting-code-review | Prepare code for review |
| receiving-code-review | Process review feedback |
| verification-before-completion | Final checks before marking done |
| finishing-a-development-branch | Branch cleanup and merge prep |
| using-git-worktrees | Git worktree workflow |
| writing-skills | Create new skills |

## Install

```bash
git clone <this-repo-url>
cd kiro-superpower-exp
chmod +x install.sh
./install.sh
```

Restart kiro-cli and press `Ctrl+Shift+S` to activate the Superpowers agent.

## Uninstall

```bash
rm -rf ~/.kiro/skills/superpowers
rm ~/.kiro/agents/superpowers.json
```
